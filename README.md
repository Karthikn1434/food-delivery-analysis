# 🍔 Food Delivery Analytics System

## 📌 Project Overview

The Food Delivery Analytics System is an end-to-end Data Analytics project that analyzes a real-world food delivery dataset to generate meaningful business insights. The project covers data cleaning, feature engineering, exploratory data analysis (EDA), SQL analysis, statistical analysis using NumPy, and data visualization using Matplotlib.

---

## 🎯 Problem Statement

Food delivery companies generate a large amount of data every day. This project analyzes customer orders, restaurant performance, delivery operations, and revenue trends to help businesses make data-driven decisions and improve operational efficiency.

---

## 🎯 Project Objectives

- Analyze a real-world food delivery dataset.
- Perform data cleaning and feature engineering.
- Identify top-performing restaurants based on revenue and orders.
- Analyze customer ratings, discounts, and delivery performance.
- Identify peak ordering hours, days, and months.
- Perform statistical analysis using NumPy.
- Solve business problems using SQL queries.
- Create visualizations using Matplotlib.
- Generate meaningful business insights for better decision-making.

## 📈 Key Insights

- Aura Pizzas generated the highest total revenue.
- Aura Pizzas received the highest number of customer orders.
- Tandoori Junction recorded the highest Average Order Value.
- Masala Junction received the highest average customer rating.
- Peak ordering time was 8 PM.
- Saturday recorded the highest number of orders.
- November received the highest number of orders.
- The Chicken Junction had the highest rider wait time.
